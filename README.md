# Lista-Zakup-w

W projekcie zostały użyte metody .append() i .prepend()
.append() - daje wybrany selektor na koniec listy
.prepend() - daje wybrany selektor na początek listy

metody .remove() i .empty() działają następująco:

.remove() - usuwa wszystki w danym elemencie jak i również sam element
.empty() - usuwa wszystko co jest zawarte w elemencie

Metody .text() i .html() w prostym języku działają tak:

.html() - szczytuje wszystko co się znajduje w danym elemencie w html
.text() - szczytuje cały tekst znajdujący się w elemencie pomijając selektory
